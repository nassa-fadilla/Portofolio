Tutorial: Deploy Website to GitHub

Tutorial Deploy Web ke GitHub

Tutorial ini bertujuan untuk membantu Anda memahami cara mengunggah project ke GitHub. Langkah-langkah ini mencakup pembuatan repositori baru, mengunggah file project, serta mengaktifkan GitHub Pages agar project dapat diakses secara online.


Langkah-Langkah:

    1. Pastikan bahwa anda telah memiliki akun github.
    2. Buat "New Repositori" dengan memasukan "nama" repository dan "public" repository.
    3. Klik button "Create Repository" dan dapatkan "Code inisialisasi Git" untuk deploy web anda ke github.
    4. Masukkan "Code Git" satu persatu pada terminal project anda, pastikan bahwa semua tahap berhasil tanpa eror/fatal.
    5. Ketika semua "Code Git" berhasil di-konfigurasi pada project anda, kembali lagi ke GitHub.
    6. Pastikan bahwa project anda telah ter-unggah pada repository github anda.
    7. Untuk mendapatkan URL live project, masuk ke menu setting pada GitHub.
    8. Masuk ke menu "Pages", pilih "Branch: main" dan klik "Save".
    9. Tunggu hingga proses deploy selesai, cek secara berkala pada setting -> pages.
    10. Setelah proses deploy setelai, dapatkan URL live Project anda pada Github Pages.

Code inisialisasi Git:

    1. git init
    2. git add .
    3. git commit -m "first commit"
    4. git branch -M main
    5. git remote add origin https://github.com/"namaakungithub"/"namarepository".git
    6. git push -u origin main
